Should you learn TypeScript?

January 15, 2018

What is TypeScript anyway? Should you know/learn TypeScript?

TypeScript (TS) is a typed super-set of JavaScript (JS) which compiles to plain JavaScript. TypeScripts adds some new ECMAScript features too. The type system of TS is relatively rich and includes: interfaces, enums, hybrid types, generics, union & intersection types, access modifiers and much more. Check the official TS page out to learn more about TypeScript. All JS code is valid TS code. You can almost rename your .js files to .ts files & start using TS. TS files are compiled to readable JavaScript, so that migration back is possible and understanding the compiled TS is not hard at all. This way TS builds on the successes of JS while improving on its weaknesses. The typing part of TS is optional but very strong IMO. It just makes JS more robust. And well, even the creator of Ruby, Yukihiro Matsumoto, said static typing could be coming to version 3.0. So you see, it is like a best practice, which might, just might be spreading.

About time & energy to learn & understand TS: honestly, if you know JS, then TS is just some hours away. It is like a 500m sprint (never seen a 500m sprint 😀). You are done with 400, another 100 to go. Then it is all about practice & including it in your projects & work. BUT if you don’t know JS, well then, it is up to you to learn JS and I would suggest learning the DOM (Document Object Model) API too. JS is incomplete without it.

TS is not going away anywhere anytime soon. It is only growing. Not only front-end software engineers must know JS but TS is an awesome plus point on the resume. In my opinion, Microsoft has done a great job with it. Then, there is Google. Google’s Angular heavily relies on TS. Learn TS & you can probably head towards Angular.

There are some awesome resources on the web to learn TS without having to pay anything at all, unless you want to. Check the official page & docs itself. Then there free as well as paid courses on Udemy. I wouldn’t pay at all to learn something like TS, specially if you already know JS. Some great youtube videos which cover the basic as well as apps in Angular to learn more TS & Angular itself.
